Flexbox Visual Guide

Understanding the axes: In flexbox, there are two axes - the main axis (red horizontal line) and cross axis (red vertical line). The direction of these axes changes based on flex-direction.

flex-direction changes the direction of the main axis. Options include: row (default), row-reverse, column, column-reverse.

flex-wrap controls whether items wrap to new lines when they run out of space. Options include: nowrap (default), wrap, wrap-reverse.

The order property controls the order in which items appear, regardless of their order in the HTML.

flex-grow (Growing to Fill Space)

Main Axis
Cross Axis
1 (no grow)
2 (grow: 1)
3 (grow: 2)

flex-grow determines how much an item can grow relative to other items. A higher value means it will take more of the available space.
